# Page Templates Spec

This spec outlines the page templates included in the PBWF framework.

## 1. Homepage Template (`Home.tsx`)
* **Key Layout Areas:**
  - Dynamic Hero: Split-screen visual background with layered dark-gradient overlays and typographic text overlays.
  - Multi-tab Search Pillar: Input fields mapping filters by criteria (crop, category, region).
  - Featured Cards: 4-column product grid with layout badges.
  - Trust Indicators: Highlight metrics (years of operation, count of dealers, farmers reached).
* **Replaceable Content:** Background image, tab options, statistics, and highlight products.

## 2. About Us Template (`About.tsx`)
* **Key Layout Areas:**
  - Brand Introduction: Split layout highlighting welcome messages and company facilities.
  - Value Props: 3-column feature blocks highlighting pillars (Trust, Speed, Innovation).
  - Accordion Block: Multi-panel accordion details for infrastructure and laboratory specifications.
  - Director Quotes: Styled quote callout cards.
* **Replaceable Content:** Copywriting statements, value images, and executive quotes.

## 3. Product Catalog Template (`Products.tsx`)
* **Key Layout Areas:**
  - Header Copy: Page titles and subtitle details.
  - Search and Filter Strip: Live text search input paired with horizontal category pills.
  - Catalog Grid: Responsive columns listing item cards.
  - Interactive Detail Modal: Springs open on card click to show packings, descriptions, and CTAs.
* **Replaceable Content:** Products dataset array and detailed specs mapping.

## 4. Social & Community Template (`Social.tsx`)
* **Key Layout Areas:**
  - Video Display: Video showcases with modal overlays for YouTube players.
  - Trials Carousel: Smooth horizontal trial photos carousel.
  - Interactive Counter: Counters showing reach and followers.
* **Replaceable Content:** YouTube links and trial photo references.\n
