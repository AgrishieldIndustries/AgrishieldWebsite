# Content Management Guide

This guide maps all content areas across the website, allowing editors to update copy without breaking design layouts.

## Page Content Mapping

### 1. Homepage (`src/pages/Home.tsx`)
* **Hero Section (Lines 180-215):** Update hero headlines, sub-taglines, and background image source URL.
* **Featured Grid (Lines 8-53):** Customize the `PRODUCTS` data array to display showcase catalog cards.
* **Trust Metrics (Lines 299-319):** Edit counts, years, and values inside the stats grid.

### 2. About Us (`src/pages/About.tsx`)
* **Core Introduction (Lines 110-151):** Customize the main paragraphs describing the company history and facility details.
* **Value Cards (Lines 23-39):** Update descriptions and icon references for value pillars.
* **Infrastructure List (Lines 41-55):** Edit title fields and descriptions inside the `INFRASTRUCTURE` dataset.
* **Director Quotes (Lines 357-418):** Customize executive statements and change image pointers.

### 3. Product Catalog (`src/pages/Products.tsx`)
* **Product Catalog Array (Lines 18-899):** Update the master list of all products including ID, name, packing sizes, prices, images, and category tags.
* **Product Descriptions (Lines 901-931):** Update detailed information maps shown inside the click modal.\n
